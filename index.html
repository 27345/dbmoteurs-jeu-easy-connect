Volvo Penta Easy Connect

Connectivité avec votre appareil mobile, Interactions simplifiées avec votre concessionaire, Accès facile aux informations moteur sur le bateau comme à la maison, Planification de vos sorties en mer.

Avec Easy-connect, vous pouvez avoir accès aux données de votre moteur directement sur votre téléphone ou votre tablette. Grâce à la fonction bluetooth, les données de votre bateau sont directement partagées sur votre appareil connecté. La vue capitaine permet de suivre les données en temps réel comme le régime moteur, la tension de la batterie, le compteur d’heures, les témoins et alarmes, la pression d’huile ou encore le taux de consommation. La vue maison, elle, permet de voir toutes les informations stockées lors de votre dernière navigation depuis chez vous, parfait pour préparer de futurs trajets.

L'application est disponible dans l'App Store et sur Google Play. L'application Easy Connect est disponible pour smartphone ou tablette Android et Apple. Pour les appareils Android, la version requise pour le système d'exploitation est 5.0 ou ultérieure avec Bluetooth® 4.2 et pour les appareils Apple, la version requise est iOS 10 ou ultérieure.

Les Fonctionalités

NMEA 2000

NMEA 2000

L’appareil se branche sur votre moteur et sur votre réseau NMEA2000 pour vous fournir les informations de votre moteur. Toutes les données venant d’autres capteurs et fonctionnalités montées à bord en NMEA2000 seront disponibles sur l’interface Easy Connect.

Vue à bord

Vue à bord

La Vue Capitaine vous permet de voir en temps réel toutes les informations essentielles. Vous pourrez paramétrer un tableau de bord personnalisé ou naviguer entre les différents onglets pour visualiser la vitesse du bateau, le régime moteur, les cartes, etc…

Vue maison

Vue maison

La Vue Maison vous permet d’avoir accès aux principales données du bateau comme le niveau du réservoir de carburant et le nombre d’heures moteur mais aussi de voir vos derniers trajets, des informations pratiques pour préparer sa prochaine sortie en mer. Il est aussi possible de partager par mail toutes les informations à votre concessionnaire pour préparer une intervention rapidement, avec toutes les informations nécessaires. Vous pouvez aussi voir vos dernières sorties en mer pour les revivre ou pour les partager sur les réseaux sociaux.

Personalisation

Tableau de bord personalisable

Les tableaux de bord sont personnalisables. Ce qui donne la possibilité d’avoir une interface sur mesure avec les données de votre bateau qui comptent le plus pour vous.

Interactions simplifiées

Communications avec votre agent Volvo Penta

L’interface Easy connect vous permet d’informer facilement votre concessionnaire, notamment avec des codes de diagnostic, le numéro de série ou le nombre d’heures du fonctionnement de votre moteur. Ces informations permettent de préparer au mieux un rendez-vous pour une intervention. Votre concessionnaire pourra ainsi vous aider plus facilement.

Partager

Partager ses sorties

Partagez vos sorties en mer avec vos amis et vos proches. Vous pourrez partager votre position ou vos itinéraires enregistrés par SMS, MMS et sur les réseaux sociaux (connexion internet requise).
